🏥 Mental Health Care Administrative & Financial Management System (MHC-AFMS)
📌 Project Overview

The Mental Health Care Administrative & Financial Management System (MHC-AFMS) is a web-based application designed to manage mental healthcare operations efficiently. It supports appointment booking, doctor scheduling, medical records, billing, and payment management with role-based access control.

This project is developed as an academic system, following real-world hospital workflows while using simulated payment processing for demonstration purposes.

👥 User Roles

Admin – Manages doctors, staff, schedules, and financial reports

Doctor – Handles appointments, diagnoses, test suggestions, and follow-ups

Staff – Assists with appointments, tests, and billing verification

Patient – Registers, books appointments, views prescriptions, invoices, and payments

✨ Key Features

Online appointment booking with time-slot management

Automated billing and invoice generation

Simulated payment workflow (no real transactions)

Doctor-wise, test-wise, and date-based financial reports

Role-based dashboards

ERD, DFD, and Function Point Analysis included

🛠️ Technology Stack

Backend: ASP.NET MVC (C#)

Frontend: HTML, CSS, Bootstrap

Database: MySQL (XAMPP)

Diagrams: ERD, DFD, Sequence Diagrams

Analysis: Function Point Calculation

⚠️ Payment Note

Payments in this system are simulated for academic and security reasons. No real payment gateway is integrated.
